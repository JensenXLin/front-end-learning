HTML属性

属性总是以 name="value" 的形式写在标签内，name 是属性的名称，value 是属性的值。

一些常用的属性名

id属性为元素指定唯一标识符

class属性为元素指定一个或者多个类名

style属性为元素直接添加CSS样式

title属性为元素提供额外提示信息，通常在鼠标悬停时候显示

HTML链接

href定义打开的目标链接

target定义链接的打开方式

rel定义链接与目标页面的关系

download提示浏览器下载链接目标而不是导航到该目标

title定义链接的额外信息，当鼠标悬停在上时的工具提示

id：用于链接锚点，通常在同一页面中跳转到某个特定位置

hreflang: 指定链接的目标URL的语言

type: 指定链接资源的MIME类型

style: 直接在元素上定义CSS样式

class: 用于指定元素的类名（CSS中定义）

空链接

href等于一些特定值的时候有特定效果，如刷新页面等

HTML头部

使用 base 定义页面中所有链接默认的链接目标地址。

使用 meta 元素来描述HTML文档的描述，关键词，作者，字符集等

使用 style 直接对HTML文件运用CSS样式
